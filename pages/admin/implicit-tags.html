Implicit Tags Editor

Below are the definitions for all regular implicit tags, which are tags that are implied by the existence of other tags on a file, or other properties of the file that can be used to search for it.

Implicit tags are defined by a search query. If a file matches the query for an implicit tag, that tag is implied.

Avoid creating loops! Loops happen when a tag implies another tag that then implies the first tag. Such cases will cause hangs when searching. Putting quotes around the tag name in the query to make it only match explicit tags can be used to break loops if two tagsshould imply each other.
