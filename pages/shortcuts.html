DokuStash: Tags Update

Sep 27, 2018

This update introduces file tags. Files can now have multiple tags that describe them in some way.

The search bar at the top of the page has replaced the old sidebar. For basic information about the new search bar, just click it. For in-depth information, check the new Search Guide.

Categories have been removed as tags have made them obsolete. The "categories" below are now just shortcuts to predefined search queries. There are also some new ones, like Animations and Latest.

File selections are now saved between sessions. That means if you select some files and close the tab or browser window, the same files will still be selected when you open the stash again. The selection menu has also been moved to the top and is only visible when files are selected.

Many other smaller changes have been made to improve usability, consistency, performance and looks.
